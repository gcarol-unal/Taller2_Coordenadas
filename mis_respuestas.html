4 Pregunta 4

Usando la herramienta “Project” re-proyecte municipios.shp (GCS_Bogota) a “Colombia Bogota Zone.prf”. Nombre el archivo de salida “municipios_planas” y guárdelo como stand alone feature class dentro de la File Geodatabase correspondiente.

Explique si se requiere Geographic Transformation y describa el proceso que ejecuta la herramienta.

4.1 Respuesta

Propiedad Valor
Geographic Coordinate System Bogota
WKID 4218
Authority EPSG

Prime Meridian Greenwich (0,0)
Datum D Bogota
Spheroid International 1924
Semimajor Axis 6378388,0
Semiminor Axis 6356911,946127946
Inverse Flattening 297,0

En este caso no se requiere Geographic Transformation, ya que no se está cambiando el datum (ambos sistemas están basados en Datum Bogotá).

La herramienta Project realiza:

  1. Conversión de coordenadas geográficas (latitud/longitud).
  2. Aplicación de la proyección Transverse Mercator.
  3. Transformación matemática de coordenadas angulares a coordenadas planas (metros).
  4. Creación de un nuevo feature class con el sistema proyectado.
